Penjelasan

1. Github adalah sebuah layanan yang berbasis web hosting untuk menghosting proyek-proyek open source yang menggunakan tool system revisi control git.
2. Github berfungsi untuk membantu atau meringankan para programmer baik secara perindividu,perkelompok atau oerganisasi dan peruhaan
3.  Proses melakukan SSH, Create Repository, Pull Request dan Push Request

- SSH

1. Pertama kita masuk pada akun github
2. Setelah itu klik pada profile
3. Pilih setting
4. Kemudian pilih SSH and GPG key
5. New SSH key
6. Kemudian muncul kolom title lalu title itu diisi
7. Setelah itu klik generating ssh key lalu klik kanan dan pilih new tab
8. Akan muncul coding ssh-keygen kemudian coding tersebut copy ke git bash
9. Buka aplikasi git bash lalu akan muncul seperti cmd
10. cat ~/.ssh/id\_rsa.pub
11. Selanjutnya muncul tulisan tersebut di copy dari ssh-rsa sampai .com
12. Lalu paste pada kolom key yang terdapat di github
13. Proses SSH selesai

- Create Repository

1. Pertama create di github
2. Buka windows explorer
3. Kemudian buat folder  baru
4. Lalu beri nama seperti repository yang dibuat pada github
5. Masuk ke folder tersebut lalu klik kanan dan pilih git bash
6. Setelah di klik akan muncul aplikasi git bash seperti cmd
7. Lalu ketik coding yaitu git init
8. git remote add origin dan link ssh pada github
9. selesai

- pull

1. lanjutan dari create repository
2. ketik coding
3. git pull origin master
4. selesai

- push

1. pada folder yang kita buat tadi pada create repository kita membuat folder lagi di dalamnya
2. setelah itu ketik coding
3. git add doc/ file yang ditambahkan
4. git commit –m &quot;bebas karena cuman keterangan&quot;
5. git push origin master
6. selesai
